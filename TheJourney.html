The Journey

How can the same tools and technologies used to create the iHealth application demo apply to other industries and use cases?

Here are examples of how we have applied data analytics and machine learning technologies to create valuable applications for other industries: 

Fraud Management System for Banks and Financial institutions

  • Area of application: Fraud management for a major bank in Asia
    • Real-time analysis of banking events and transactions
    • Classification of transactions (fraudulent/otherwise)
    • Designed as an extension of the AML product suite for OFS

Healthcare Media Analytics Platform (HMAP)

  • Area of Application: Enterprise data warehouse for a U.S.-based, large healthcare media company
    • Multi-feed data ingestion, routing and analytics, machine learning and natural language processing (NLP) for multi-demographic healthcare campaign management

Recommender System

  • Area of Application: E-Commerce
    • Recommends products for purchase based on customer behavior in terms of interactions and preferences
    • Provides real-time recommendations based on categories of information, context, behavior and reference data
